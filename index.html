    Tips
    Soft, dreamy fade Set both Size and Softness to 70–100%. Size controls how deep the fade reaches; Softness alone won't create a wide fade.
    Hard vignette Size 15–35%, Softness 5–20%. Keeps most of the image sharp with a tight border.
    Size vs Softness Size = how far the fade band reaches inward. Softness = how gradually it transitions. Both need to be high for a wide, gentle fade.
    Ring / halo effect Enable both Inner and Outer fade. The image becomes transparent at the edges and the center, leaving an opaque ring.
    Circle on non-square images The fade will be elliptical. Use Square shape for equal fade depth on all four sides.
    Global opacity Uniformly scales every pixel's alpha — useful for watermark overlays or compositing.
    Preview resolution By default, the live preview uses a downscaled copy for speed. Enable Full resolution preview for pixel-accurate feedback. The downloaded file is always full resolution.
